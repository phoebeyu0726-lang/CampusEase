User Story 1：攜帶重物或行動不便者的無障礙導航 
• 作為一位 需要推著實驗器材推車（或受傷暫時行動不便）的校園使用者， 
• 我想要 系統能為我規劃一條完全避開階梯、陡坡、高溫曝曬，且精準引導至大樓自動門與無障礙電梯的舒適路線， 
• 因為 現有的地圖只導航最短距離，常常讓我走到一半才發現被樓梯卡住，我需要一條真正體貼、友善且省力的無障礙幸福路徑。

User Story 2：通勤大學生規劃舒適交通方案 
• 作為一位 每天需要往返校園與住所的通勤大學生， 
• 我想要 在出門前查看下雨、高溫、空污和 YouBike 資訊，並提前規劃通勤路線與交通方式， 
• 因為 我想避開不舒適的環境，選擇最安全、方便且有效率的通勤方案。 

User Story 3：外地旅客快速適應當地交通與環境 
• 作為一位 初次來到當地的外地旅客， 
• 我想要 透過一個整合天氣、空氣品質與 YouBike 即時資訊的平台，快速了解當下環境與交通選擇， 
• 因為 我不熟悉當地路線與交通系統，希望能用最短時間規劃出安全、方便且順暢的移動方式，提升整體旅遊體驗與移動效率。
